# Customer Shopping Behavior Analysis

## 🚀 Live Demo
**(https://retail-data-insights.streamlit.app/)**


## 🔧 Tools Used
- Python (pandas, streamlit) - Data cleaning & web app
- SQL (PostgreSQL) - Business queries & analysis
- Power BI - Interactive dashboards
- Excel - Quick analysis

## 📈 Key Insights
- Subscribed customers spend 15% more per transaction
- Age group 26-35 contributes 40% of total revenue
- Clothing category drives highest purchase volume
- Top 5 products account for 30% of sales
